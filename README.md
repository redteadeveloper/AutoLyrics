# AutoLyrics

## About
 
[AutoLyrics](https://github.com/redteadeveloper/AutoLyrics) is an app which can add lyrics to mp3 files.

Created using [Electron](https://www.electronjs.org/) and [Browserify](http://browserify.org/).
